Hi, I'm Kevin, a game designer
with a passion for creating new
and unique experiences.

If you couldn't tell, I like video games.
Believe it or not, I like making them too.

I've spent years working with nearly every aspect of game design. From technical artist to programmer to 3d modeler, I've done it all. I specalise in programming and 3d modeling, and have become very familar with software such as Unity, Maya, Jira, and Photoshop. In my spare time, I like to use my skills to create projects in order to learn more and create games I am proud of.

Here's some of the things I've been working on

Gameplay & Systems Programming

I've used C# to create my own code for every project I've made. This is from a capstone game pitch titled Airwav.

3d Modeling

I typically use Maya and Photoshop for modeling and UV mapping. These are from various school projects and gamejams.

Menu & UI Design and Implementation

I have been able to use the Unity engine to create a wide variety of interactive menus and game UI. This is from a gamejam game titled Fishing at the End of the World.

2d Asset Creation

I create a wide variety of 2d assets, and because of that I use many softwares. For texturing and pixel art, I use a mixture of Photoshop and Aseprite to create my assets.

VFX & Shader Creations

I've spent a lot of time learning the ins and outs of the Unity VFX and Shader graphs, and have used them plenty to create cool visual effects. These ones are a part of my Game Design capstone project, Waves and Wizards.

Finished Projects

As I have finished projects through school and personal projects, I have made sure to make each of them available on my itch.io page. These vary in style and complexity, and allow me to show off many different styles of games. .
